            与 v2 相比，重写了 Markdown 插件的逻辑，减少 container 嵌套。

            # ECharts 插件

            让你的 VuePress 站点的 Markdown 文件支持 ECharts 图表。

            使用本插件时，需要自己安装 echarts 依赖，本插件会将 echarts 加到 optimizeDeps 中。 另外，本插件使用了 vueuse，用来做宽度自适应。

            Playground 插件 v2

            # Playground 插件 v2

            让你的 VuePress 站点中的 Markdown 文件支持 Vue SFC Playground 类似的代码案例。

            同时支持外置和内置两种模式：

            • 外置模式，通过嵌入iframe的形式来引入 playground.
            • 内置模式，通过@vue/repl直接渲染出来 playground.

            比 v1 更强大。
